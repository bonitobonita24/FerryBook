# F and S Marine Transport Inc. — Phase 2.8 Mockup

Interactive UI mockup for the Batangas ↔ Lubang Island ferry booking system.  
**40 screens** at full fidelity. Not live — no data persists.

## Stack

- React 18 + Vite 5
- Tailwind CSS (CDN)
- Lucide React icons
- Recharts (admin dashboard charts)
- Deploys to Vercel

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deploy to Vercel

```bash
npx vercel
```

Or connect the repo to Vercel and it auto-deploys on push.
